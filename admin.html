🔑 Publicação automática no site

Conecte uma vez só. Depois disso, todo produto que você adicionar, editar ou remover aqui é publicado direto no site — sem precisar baixar nem subir arquivo.

Como criar o token (uma vez só):
1. Acesse github.com/settings/personal-access-tokens/new
2. Em "Repository access" escolha "Only select repositories" → selecione beleze699oficial
3. Em "Permissions" → "Repository permissions" → mude "Contents" para Read and write
4. Clique em "Generate token", copie e cole aqui.
O token fica guardado só neste navegador/celular — se usar outro aparelho, precisa configurar de novo lá também.
